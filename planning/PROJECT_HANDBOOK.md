# Project Name

**PolicyVault** *(Working Name)*

Tagline:

**Your AI-Powered Economic Intelligence Platform**

---

# Mission

Build a fully automatic AI-powered Personal Knowledge Operating System that continuously collects, organizes, understands and presents every important official publication relevant to RBI Grade B.

The platform should become the **single source of truth** for economics, banking, finance and public policy.

The user should never need to browse multiple government websites again.

---

# Primary Goal

Instead of collecting PDFs,

collect **knowledge**.

Every official publication should automatically become:

* Searchable
* Readable
* Summarized
* Connected
* Search indexed
* AI searchable
* Revision ready

# Product Principles

## Principle 1

Official Sources First

Always prefer:

Official APIs

↓

Official RSS

↓

Official XML

↓

Official PDFs

↓

Official Pages

Only use trusted secondary sources where official sources are unavailable.

---

## Principle 2

Never Miss Anything

Every scheduled publication should be tracked.

The platform should know

Released

Expected

Delayed

Archived

---

## Principle 3

Everything Stays Inside PolicyVault

Never force the user to leave the platform.

PDFs should open inside the application.

Documents should be readable inside the application.

Search should happen inside the application.

AI should answer using documents inside the application.

# Version 1 Success Criteria

The platform is considered successful when:

* Every major official publication is automatically detected.
* New content appears without manual intervention.
* Documents can be read entirely inside the application.
* AI summaries and revision material are available.
* Search finds information across all collected knowledge.
* The dashboard clearly shows what is new, what is unread, and what is expected.
* The user can prepare for RBI Grade B using this platform as the primary knowledge source.

---

# Guiding Principle

**If a feature does not help understand, retain, revise, or connect official economic knowledge, it does not belong in Version 1.**
